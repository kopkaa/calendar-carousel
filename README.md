# Calendar


## Tech stack

- [react](https://github.com/facebook/react)  
- [tailwindcss](https://github.com/tailwindlabs/tailwindcss)

## Requirements
-  Node v22.3.0

## Run project

- run `npm i` inside root directory
- run `npm run dev` to run project
